2.5 KELLY — TAILLE DE MISE OPTIMALE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Critere de Kelly :
  f* = (P_sim * Cote - 1) / (Cote - 1) = ___%
  Kelly fractionnel (quart) : f*/4 = ___%

Interpretation :
  → Si f* < 0% : PAS DE MISE (P.sim < P.implicite)
  → Si f*/4 < 1% : value trop faible pour miser
  → Si f*/4 entre 1-2% : mise standard (1 unite)
  → Si f*/4 entre 2-3% : mise forte (si confiance elevee uniquement)
  → Si f*/4 > 3% : ALERTE — verifier les donnees, probablement surestimation
  → Max absolu 2.5% bankroll par pari

ROBUSTESSE de la value :
  - Value maintenue dans les 3 scenarios → ROBUSTE ✓
  - Value disparait a -10% seulement → FRAGILE (malus confiance)
  - Value disparait a -20% seulement → ACCEPTABLE
  - Value disparait a +10% favorable → INCOHERENT (revoir l'analyse)

| Ecart (Sim - Marche) | EV        | Verdict                  |
|----------------------|-----------|--------------------------|
| > +10%               | > +0.15   | VALUE BET FORTE ***      |
| +5% a +10%           | +0.05/15  | VALUE BET CONFIRMEE **   |
| +2% a +5%            | +0.02/05  | VALUE BET LEGERE *       |
| 0 a +2%              | 0 a +0.02 | ZONE NEUTRE              |
| < 0                  | < 0       | PAS DE VALUE             |

Confiance (/25) :
| Critere                          | Max | Score | Justification            |
|----------------------------------|-----|-------|--------------------------|
| Qualite donnees (stats)          | /5  |       |                          |
| Completude (absences, compo)     | /5  |       |                          |
| Stabilite (sanity+sensibilite)   | /5  |       |                          |
| Coherence H2H + tendances        | /5  |       |                          |
| Coherence consensus marche        | /5  |       |                          |
TOTAL : ___/25 (20-25=elevee, 15-19=moderee, <15=faible)

⚠️ MALUS CONFIANCE (cumulables) :
  - Sport exotique / peu couvert mediatiquement : -2 pts sur "Qualite donnees"
  - Division inferieure / ligue mineure : -1 pt sur "Qualite donnees"
  - Pas de stats avancees disponibles : -2 pts sur "Qualite donnees"
  - Blessures non confirmees (source unique) : -1 pt sur "Completude"
  - H2H insuffisant (<3 matchs) : -2 pts sur "Coherence H2H"
  - Divergence stats entre sources : -1 pt sur "Qualite donnees"
  - Value fragile (disparait a +/-10%) : -2 pts sur "Stabilite"
  - Source unique pour une stat cle : -1 pt sur "Qualite donnees"
  - Ecart P.sim vs P.marche > 15% sans justification forte : -2 pts sur "Coherence marche"
  - Mouvement de cotes contraire a la recommandation : -1 pt sur "Coherence marche"
