2.3 PROBABILITE — TRANSFORMATION LOGISTIQUE
delta_norm = (Score_A - Score_B) / 100
P_multi(A) = 1 / (1 + 10^(-3.5 * delta_norm))
⚠️ k=3.5 en TT car les ecarts de niveau sont souvent tres prononces.

Reference rapide (k=3.5) :
delta 0.03→54% | 0.05→58% | 0.08→64% | 0.12→70% | 0.18→78% | 0.25→87%

P_multi(B) = 1 - P_multi(A)

2.4 CROSS-CHECK CLASSEMENT ITTF
P_elo_approx(A) = 1 / (1 + 10^((rang_B - rang_A) / 200))
(Approximation : chaque 200 rangs d'ecart ITTF ≈ 200 points Elo)
Si ecart > 12% avec P_multi → signaler DIVERGENCE.

PROBABILITE FINALE :
P(A) = 0.65 * P_multi(A) + 0.35 * P_elo_approx(A) = ___%
P(B) = 1 - P(A) = ___%
(Si H2H >= 6 matchs : P(A) = 0.5 * P_multi + 0.25 * P_elo + 0.25 * P_h2h)
