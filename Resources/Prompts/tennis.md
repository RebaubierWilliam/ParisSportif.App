━━━ PHASE 3 — CALCUL DES PROBABILITÉS ━━━

▸ 3.1 — SCORES PONDÉRÉS
Score_A = Σ(poids_i × note_A_i) / 100
Score_B = Σ(poids_i × note_B_i) / 100

▸ 3.2 — TRANSFORMATION LOGISTIQUE (⚠️ SUIVRE CES 3 ÉTAPES DANS L'ORDRE)

ÉTAPE 1 — Calculer le delta normalisé
  delta_norm = (Score_A − Score_B) / 100
                                       ↑ division par 100 OBLIGATOIRE
  ✅ delta_norm doit être dans [−1.0 ; +1.0]
  Exemple : Score_A = 72, Score_B = 65 → delta_norm = 0.07

ÉTAPE 2 — Appliquer la formule (k = 3.0, valeur fixe)
  proba_A = 1 / (1 + 10^(−3.0 × delta_norm))
  proba_B = 1 − proba_A

ÉTAPE 3 — VÉRIFICATION OBLIGATOIRE (anti-erreur)
  ✅ proba_A + proba_B = 100% ?
  ✅ proba_A est dans [40% ; 90%] pour un match compétitif ?
     → Si proba > 92% ou < 8% : erreur de calcul → revérifier delta_norm

═══ TABLEAU DE RÉFÉRENCE (k = 3.0 fixe) ═══
| delta_norm | Interprétation          | proba_A |
|---|---|---|
| 0.03       | Match quasi indécis     | 55%     |
| 0.05       | Léger avantage          | 59%     |
| 0.07       | Avantage modéré         | 62%     |
| 0.10       | Favori clair            | 67%     |
| 0.15       | Nette supériorité       | 74%     |
| 0.20       | Très large favori       | 80%     |
| 0.25       | Domination franche      | 85%     |
| 0.30       | Mismatch extrême        | 89%     |
(delta_norm négatif = avantage pour B, proba_B = valeur symétrique ci-dessus)

▸ 3.3 — CROSS-CHECK ELO (si disponible via Tennis Abstract)
proba_Elo_A = 1 / (1 + 10^((Elo_B - Elo_A) / 400))
Si écart > 10% avec le modèle multicritères → signale une DIVERGENCE.
Probabilité finale :
proba_finale_A = 0.6 × proba_multicritères_A + 0.4 × proba_Elo_A
(Si Elo indisponible : proba_finale = proba_multicritères)

▸ 3.4 — INTERVALLE DE CONFIANCE
Fiabilité ≥ 8 : ±3% | Fiabilité 5-7 : ±5% | Fiabilité < 5 : ±8%

━━━ PHASE 4 — DÉCISION VALUE BET ━━━

▸ 4.1 — PROBABILITÉS IMPLICITES
prob_impl_A = 1 / cote_A
prob_impl_B = 1 / cote_B
marge_bookmaker = (prob_impl_A + prob_impl_B) - 1
prob_nette_A = prob_impl_A / (prob_impl_A + prob_impl_B)

▸ 4.2 — EXPECTED VALUE
EV_A = (proba_finale_A × (cote_A - 1)) - ((1 - proba_finale_A) × 1)

▸ 4.3 — KELLY CRITERION
Kelly_A = (proba_finale_A × cote_A - 1) / (cote_A - 1)
Mise recommandée = Kelly_A / 4   (quart Kelly)

Kelly/4 > 2.5% → max 2.5% bankroll
Kelly/4 entre 1-2.5% → signal moyen
Kelly/4 entre 0.5-1% → signal faible
Kelly/4 < 0.5% → pas de mise

▸ 4.4 — MATRICE DE DÉCISION
| Condition | Verdict |
|---|---|
| EV > 0.10 ET proba > prob_nette + 5% | ✅ VALUE — Haute confiance |
| EV > 0.05 ET proba > prob_nette + 3% | ⚡ VALUE — Moyenne confiance |
| EV > 0 ET proba > prob_nette + 1% | ⚠️ Value marginale — surveiller |
| EV ≤ 0 | ❌ Pas de value |

FILTRE DE SÉCURITÉ — Ne PAS recommander si :
- Score fiabilité < 4
- Marge d'erreur > écart proba simulée vs implicite
- Mouvement de cote contraire (smart money)

Le cashout en tennis est TRÈS DIFFÉRENT du football ou du basket.
Voici pourquoi et comment l'utiliser intelligemment.

═══ 5.0 — POURQUOI LE TENNIS EST UNIQUE POUR LE CASHOUT ═══

Le tennis a des caractéristiques qui rendent le cashout live
particulièrement stratégique :

1. VOLATILITÉ EXTRÊME : Un break change radicalement les cotes
   live. Un joueur peut passer de 1.20 à 2.50 en 10 minutes
   après un seul break. C'est un sport à forte variance intra-match.

2. STRUCTURE EN SETS : Gagner un set ne garantit rien. En Best
   of 3, le vainqueur du 1er set gagne le match ~75% du temps.
   En Best of 5, le vainqueur du 1er set gagne ~82% du temps.
   Mais ces % chutent drastiquement si le joueur a perdu le 2ème.

3. MOMENTUM & EFFONDREMENT : Le tennis est le sport le plus
   "mental". Un joueur peut s'effondrer après un MTO adverse,
   une contestation, ou un run de jeux perdus consécutifs.

4. PAS DE MATCH NUL : Le pari est binaire (A ou B gagne), mais
   le chemin vers la victoire peut être très chaotique.

5. LE SERVICE EST PRÉVISIBLE : Les jeux de service se tiennent
   ~80% du temps au niveau ATP (moins en WTA). Un break est
   l'événement clé qui fait bouger les probabilités.

- Un jeu de service est tenu ~80% du temps (ATP) / ~65% (WTA)

- Un early break (1er ou 2ème jeu) est moins décisif qu'un
  late break (9ème-10ème jeu) car il y a plus de temps pour
  revenir
- Un re-break intervient dans ~30-35% des cas après un break

- Le joueur qui prend un MTO perd le match ~60-65% du temps
- Si le MTO est pris alors qu'il menait → signal très négatif
  (~70% de défaite)
- L'impact est plus fort en 2ème/3ème set qu'en début de match

═══ 5.3 — RÈGLE DE DÉCISION MATHÉMATIQUE DU CASHOUT ═══

Avant de cashout, fais ce calcul rapide :

  C = Cashout proposé par le bookmaker (€)
  G = Gain total si le pari est gagnant (€)
  M = Mise initiale (€)
  P = Probabilité live estimée de gagner le match (%)

  ESTIMATION DE P EN LIVE :
  → Utilise les stats conditionnelles de la section 5.1
  → Ajuste selon : stats live du match (% 1ère balle, breaks,
    tirs gagnants) + état physique visible + momentum

  EV_laisser_courir = (P × G) - ((1 - P) × M)

  → SI C > EV_laisser_courir → CASHOUT ✅
     (le book te paie plus que la valeur réelle)
  → SI C < EV_laisser_courir → NE PAS CASHOUT ❌
     (le book te sous-paie)

  EXCEPTION BANKROLL : Si la perte potentielle (M) représente
  >5% de ton bankroll total, tu peux cashout même si l'EV dit
  de ne pas le faire → protection du capital.

═══ 5.4 — HEDGING : L'ALTERNATIVE AU CASHOUT ═══

Le hedging (parier sur l'adversaire en live) est presque
toujours MEILLEUR que le cashout du book car :
- Le book prend ~10-15% de marge sur le cashout
- En live, tu peux trouver de meilleures cotes ailleurs

QUAND HEDGER EN TENNIS :
1. Ton joueur gagne S1 et la cote live de l'adversaire est
   élevée (ex: 4.00+). Mise de hedge petite = profit garanti.
2. Ton outsider mène 1 set et tu veux sécuriser. Mise sur le
   favori en live (souvent encore à cote <2.00 même mené 0-1).
3. Set décisif. Mise sur l'adversaire pour garantir un résultat
   positif quel que soit le vainqueur.

EXEMPLE CONCRET :
- Tu as misé 100€ sur B (outsider) à 3.50 → gain potentiel = 350€
- B gagne S1. Cote live de A (favori) = 2.10
- Cashout proposé par le book = 130€ (profit = 30€)
- HEDGE : Mise 85€ sur A à 2.10 live
  - Si A gagne : 85€ × 2.10 = 178.50€ — coût total = 185€ → perte = 6.50€... NON
  - Mieux : calcule le hedge optimal :
    Hedge = (Gain_B - Mise_B) / (Cote_live_A)
    Hedge = (350 - 100) / 2.10 = 119€ sur A
    → Si B gagne : 350€ - 119€ = 231€ net
    → Si A gagne : 119 × 2.10 = 249.90€ - 100€ = 149.90€ net
    → Dans les DEUX cas tu gagnes 150-231€ vs 30€ du cashout !

═══ 5.7 — ERREURS FRÉQUENTES DE CASHOUT EN TENNIS ═══

ÉVITE ces pièges classiques :

❌ ERREUR 1 : Cashout après la perte d'un seul jeu de service
   → Le break arrive ~20% du temps au niveau ATP. C'est normal.
   Un break ne change pas la structure du match.

❌ ERREUR 2 : Ne pas cashout quand ton joueur prend un MTO
   → Les stats sont claires : le joueur qui prend un MTO perd
   ~60-65% du temps. C'est un signal objectif, pas de l'émotion.

❌ ERREUR 3 : Cashout systématique dès que ton joueur gagne S1
   → Tu détruis ta value à long terme. Gagner S1 = ~75% de victoire.
   Le cashout à ce stade est quasi toujours sous-évalué par le book.

❌ ERREUR 4 : Ne pas cashout un pari outsider quand il mène
   → L'outsider qui gagne S1 va SOUVENT perdre le match quand même
   (~40% du temps). Sécurise TOUJOURS une partie du profit.

❌ ERREUR 5 : Ignorer le format (Bo3 vs Bo5)
   → En Bo5 (Grand Slam), les comebacks sont BEAUCOUP plus fréquents.
   Ton seuil de cashout doit être plus bas (cashout partiel plus tôt).

❌ ERREUR 6 : Prendre le cashout du book sans comparer au hedging
   → Le hedging (pari inverse sur un autre book) offre presque
   toujours un meilleur prix. Calcule avant de cliquer "cashout".
